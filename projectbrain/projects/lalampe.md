# LaLampe Status

## Laatst bijgewerkt
2026-05-02

## Strekking
LaLampe is een workshopproject rond het transformeren van bestaande objecten tot persoonlijke lampen. De nadruk ligt op ervaring, proces, materiaalkeuze, eenvoudige techniek en transformatie. De focus ligt vooral op het testen van een kleine avondworkshop met nadruk op ervaring en transformatie, niet op agressieve verkoop.

## Huidige fase
Voorbereiding / testfase. De eerste logische vorm is een kleine avondworkshop met een beperkt aantal deelnemers.

## Laatste ontwikkeling
De Projectbrain-structuur wordt opgezet zodat LaLampe-status vanuit ChatGPT-brainstorms via `PB` kan worden vastgelegd in GitHub.

## Beslissingen
- LaLampe krijgt een eigen Projectbrain-statusbestand.
- Projectstatus moet niet afhankelijk zijn van handmatige nette documentatie.
- `PB` wordt de korte trigger om een brainstorm door te passen naar GitHub.

## Open acties
- Definitieve workshopflow uitschrijven.
- Materiaallijst en veiligheidscheck afronden.
- Prijs en groepsgrootte definitief maken.
- Voorbeeldfoto's en procesfoto's maken.
- Landingspagina / marketingtekst aanscherpen.

## Risico's / onduidelijkheden
- De status wordt pas echt automatisch bijgewerkt zodra de Projectbrain API en GPT Action gekoppeld zijn.
- Brainstorm-inhoud uit gewone ChatGPT-chats moet expliciet via `PB` worden doorgestuurd.

## Eerstvolgende stap
Projectbrain API testen met een LaLampe-input in `check` mode en daarna koppelen aan een GPT Action.
