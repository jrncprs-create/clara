# Clara 3.0

Clara 3.0 is de nieuwe, parallelle werkruimte voor Clara als gedeelde operating layer.

Deze map is bewust losgezet van de bestaande root-app.

## Doel

Eerst voelen of Clara 3.0 klopt als gedeelde cockpit voor:

- Jeroen
- Marlon
- later Joep

## MVP

Dashboard V1 bevat alleen:

- Vandaag
- Projecten
- Waiting For
- Team Activity
- Focus

## Nog niet

- geen Supabase
- geen API
- geen AI-parser
- geen mail
- geen agenda
- geen automatische workflows

## Belangrijk

De bestaande Clara blijft voorlopig legacy/stable.

Clara 3.0 wordt hiernaast opgebouwd zodat niets stukgaat.
